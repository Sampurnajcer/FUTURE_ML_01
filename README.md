📊 Sales Forecasting Dashboard

📌 Project Overview
This project is a Machine Learning-based Sales Forecasting Dashboard built using Streamlit. It predicts future sales using historical data and Linear Regression.

---

🛠️ Tools & Technologies
- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-learn
- Streamlit

---

📂 Dataset
The dataset used is a Superstore sales dataset containing:
- Order Date
- Sales
- Other business-related features

---

⚙️ Features
- Data loading and cleaning
- Monthly sales visualization
- Machine Learning model (Linear Regression)
- Model evaluation (MAE, MSE, RMSE)
- Interactive Streamlit dashboard
- Future sales prediction

---
